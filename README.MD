# NGINX with SSL, GEOIP, STREAM, VTS, DYNAMIC UPSTREAM, RTMP modules

## Notes
* You can run `centos-build-nginx.sh` localy on your CentOS 7 (7.4 tested) to compile NGINX (run script from root)
